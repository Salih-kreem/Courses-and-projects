[1] Browser Object Model:- BOM
    
  - Window Object Is the browser window.
  - Window contain Document Object Model.
  - Test Document and console.
  - All global variables and objects and functions are members of 
    window object.
    
1- What we can do with window object ?
  - Open window
  - Close window.
  - Move window.
  - Print document.
  - Resize window.
  - Run code after period of time once or more.
  - Fully control the url.
  - Save data inside browser to use latter.
  

[2] SetTimeout And ClearTimeout:-
    
  1- SetTimeout: You can use function and anonymous function.
     setTimeout(function, timeout, additional prameters)
     
  2- Clear Timeout: Can stop function in the time wating before start.
     clearTimeout(Identifier or number setTimeout);
     
    Example:
     
     let a = setTimeout(function() {
     console.log("Good morning");
     },5000);
     
     btn.onclick = () => {
     clearTimeout(a);
     };

  [7] Local Storage:-
      
      Part of the web storage API are three tools to save key/value pairs locally.
      
      1- setItem().
      - localStorage.setItem("key", "value");
      - localStorage.key = "value";
      - localStorage["key"] = "value";
      
      2- getItem();
      - localStorage.getItem("key");
      - localStorage.key;
      -localStorage["key"];
      document.body.style.background = localStorage['color or key'];
      
      3- removeItem();
      - localStorage.removeItem('key');
      
      4- clear();
      - localStorage.clear();
      
      5- key(); take index key.
      - localStorage.key(index);
      
      6- JSON.stringify(); تحول البيانات من مصفوفة او كائن الى نص
      - localStorage.setItem(JSON.stringify('key', 'value'));
      
      7- JSON.parse(); تحول البيانات من نص الى مصفوفة او كائن
      - JSON.parse(localStorage.getItem('key'));
